A single-file status dashboard for my home server. One HTML page, no dependencies, polls a tiny JSON endpoint and shows uptime, disk, and temperatures at a glance.

Built because every existing dashboard wanted a database and a login screen to tell me whether three laptops were awake. This one is 400 lines and lives in a bookmark.
